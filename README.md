# ⚡ Energy Digital Twin using RL & Metaheuristics

## 📌 Overview

This project implements a **Digital Twin for Smart Energy Optimization** using:

* Reinforcement Learning (RL)
* Genetic Algorithms (GA)
* Real-world energy dataset
* Dockerized microservices architecture

---

## 🏗️ Architecture

* 🐍 Python Simulation Engine
* 🛢️ MySQL Database
* 📊 Grafana Dashboard
* 📡 Prometheus Monitoring
* 🐳 Docker Containers

---

## 🚀 Features

* Real-time energy simulation
* Dataset-driven modeling
* Optimization using RL & GA
* Live dashboard visualization
* Scalable containerized deployment

---

## ⚙️ Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/ShaqilShaheen10/energy-digital-twin.git
cd energy-digital-twin
```

### 2. Run with Docker

```bash
docker compose up --build
```
